# 📂 Universal File Organizer

An automated Python script that instantly declutters your messy folders by sorting files into neat, categorized directories based on their extensions. 

Built as a smart, time-saving utility for students and professionals! 

⭐ **If you find this useful, please consider giving the repo a star!** ⭐

## ✨ Features
* **Instant Sorting:** Moves PDFs to Documents, MP4s to Videos, PNGs to Images, and more.
* **Safe Execution:** Only moves files, never deletes or overwrites them.
* **Beginner Friendly:** Uses only built-in Python libraries (no complex installations required).

## 🚀 How to Use It

1. Make sure you have Python installed on your system.
2. Download or clone this repository to your computer.
3. Open `organizer.py` in any text editor.
4. Scroll to the bottom and change the `target_directory` variable to the folder you want to clean (like your Downloads folder). Remember to use forward slashes `/`!
5. Open your terminal or command prompt and run the script:
   ```bash
   python organizer.py
